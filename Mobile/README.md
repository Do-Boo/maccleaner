# MacCleaner Mobile

iOS 17 이상을 대상으로 하는 iPhone용 MacCleaner 동반 앱입니다.

## 기능

- **기기 상태**: 전체·여유 저장 공간, 배터리, 네트워크 연결 상태
- **사진 정리**: 스크린샷, 60초 이상 동영상, 동일 해상도와 가까운 촬영 시간대의 유사 후보
- **파일 보관함**: 파일 앱에서 가져오기, 크기·수정일 확인, 공유, 삭제

유사 촬영 후보는 완전한 중복 판정이 아닙니다. 사용자가 사진을 직접 확인한 후 선택해야 하며, 삭제된 사진은 Photos의 최근 삭제된 항목으로 이동합니다.

## 프로젝트 생성

프로젝트 파일은 `project.yml`을 기준으로 XcodeGen이 생성합니다.

```bash
cd Mobile
xcodegen generate
open MacCleanerMobile.xcodeproj
```

## 실기기 실행

1. Xcode에서 `MacCleanerMobile` 타깃을 선택합니다.
2. Signing & Capabilities에서 본인의 개발 팀을 지정합니다.
3. 연결된 iPhone을 실행 대상으로 선택합니다.
4. 사진 정리 화면에서 사진 보관함 접근을 허용합니다.

파일 가져오기는 시뮬레이터에서도 동작하지만, 사진 보관함 분류와 삭제는 실제 사진이 있는 기기에서 검증하는 것이 정확합니다.

## 제한 사항

iOS는 앱별 샌드박스를 사용하므로 다음 작업은 지원할 수 없습니다.

- 다른 앱의 캐시 또는 데이터 삭제
- 시스템 로그와 임시 파일 직접 정리
- 설치된 다른 앱의 제거 또는 시작 항목 관리

모바일 앱은 Photos 권한으로 노출된 항목과 앱의 Documents 영역만 변경합니다.
